# cad

This folder contains the CAD designs for the chessboard.

You will need to 3D print all the items in `3d-prints/` - the board has been split into parts so it fits on print beds. 

You will also need to laser cut the base and spacer from `laser-cut/`.

The chessboard was designed in TinkerCAD but has been exported to Fusion 360 .f3d for editing if required.
